Project Display
This webpage displays every educational project I have done throughout my time pursuing The Odin Project. These taught me HTML, CSS, and JS. Each project has a brief description and overview of what I learned. As I continue with The Odin Project, I plan on adding any completed projects to this list, and possibly updating some of these into a better version in the future. I plan on leaving the current versions as well, as I like seeing where I came from. Projects are displayed from newest to oldest.
Calculator
A web calculator that can do simple operations
This project combined many areas of the course so far, involving JavaScript logic, updating DOM elements, and the visual elements of the webpage. I mostly focused on making it hard to break. I tried to make sure it displayed both digits after a decimal point and numbers in exponential form correctly.
Etch a sketch
A page with cells that can be colored by moving a mouse over them
This project taught me the use of event listeners other than click, and how to generate many new DOM elements with JavaScript
Rock Paper Scissors
JavaScript implementation of a rock paper scissors game using HTML elements for input
I created JavaScript functions to run a game, and learned how to get user input through prompt. In a later project, I linked the game functionality with buttons on the webpage that would call the game functions when clicked.
Shopping List
Webpage to test JavaScript interacting with html elements
I learned how to get references to HTML elements with JavaScript, and how to modify the DOM. I added, removed, and learned how to modify DOM elements.
Personal Website
HTML and CSS layout for a personal website homepage
This project taught me the flexbox method to arrange elements on a page. I learned how the axes work and some of the aspects to make space be distributed how I wanted it. It was immediately interesting, as what I had done for the recipe project would have been much easier with flexbox.
Recipe webpage
HTML and CSS documents for a mock recipe website.
I learned basic HTML to create pages and link them. I used the basic types of boxes available and methods to mark components for CSS. I learned how to safely add external links, and how to add images with backup text.
